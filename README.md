# literAlura

### Requirements
Start building a project in https://start.spring.io/
- Java 17
- Spring Boot 3.3.1
- Package manager: Maven
- Postgresql >= 16.0

#### Dependencies
- PostgreSQL Driver
- Spring Data JPA
- Jackson >= 2.16 
- We'll be using the free book provider API [Gutendex](https://gutendex.com/)
